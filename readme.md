# 深度学习示例项目（Deep Learning Examples）

> 用于系统性整理与规划深度学习各业务方向的示例代码与实验。当前仅提供**全面的大纲**（不含实现细节）。

---

## 1. 项目目标与范围

- 覆盖主流深度学习业务方向：计算机视觉、NLP、推荐系统、音频/语音、视频、多模态、时序预测等  
- 为每个方向提供可扩展的子任务目录与示例入口  
- 统一数据、训练、评估、部署与工程化规范

---

## 2. 目录规划（建议）

- `cv/`：计算机视觉
- `nlp/`：自然语言处理
- `rec/`：推荐系统与广告
- `audio/`：音频与语音
- `video/`：视频理解
- `multimodal/`：多模态（图文/音视频/跨模态）
- `timeseries/`：时间序列与预测
- `graph/`：图神经网络（GNN）
- `rl/`：强化学习
- `generative/`：生成式建模（扩散/自回归等）
- `tooling/`：工程化与通用组件（数据管道/训练框架/评测/部署）
- `docs/`：文档与实验记录

---

## 3. 计算机视觉（CV）

### 3.1 基础任务
- 图像分类（单标签/多标签）
- 目标检测（通用/小目标/旋转框）
- 语义分割
- 实例分割
- 关键点检测/姿态估计
- 目标跟踪（单目标/多目标）

### 3.2 增强与低层视觉
- 超分辨率
- 去噪/去雨/去雾
- 去模糊
- 图像修复/补全（Inpainting）
- 风格迁移
- 图像质量评估（IQA）

### 3.3 OCR 与文档智能
- 文本检测
- 文本识别
- 版面分析（Layout）
- 表格识别
- 关键信息抽取（KIE）

### 3.4 3D 与几何视觉
- 深度估计
- 立体匹配
- 点云分类/分割/检测
- NeRF/3D 重建（目录预留）

### 3.5 工业与场景应用
- 缺陷检测
- 异常检测（重建式/对比学习/密度估计）
- 人脸（检测/识别/活体/属性）
- 细粒度识别
